# Libft - Your First C Library

## Description

The goal of this project is to create our own C library, `libft.a`, which gathers a collection of highly useful standard functions that we will be allowed to use throughout our curriculum at 42. Since we are restricted from using most standard library functions in future projects, this project helps us deeply understand how they work under the hood by rewriting them from scratch. It serves as a foundational toolkit for all upcoming C assignments.

## Library Description

The library includes a variety of functions which can be categorized into three main parts:

### Part 1: Libc Functions

Rewrites of standard C library functions to understand memory and string manipulation:

* **Character Checks/Conversions:** `ft_isalpha`, `ft_isdigit`, `ft_isalnum`, `ft_isascii`, `ft_isprint`, `ft_toupper`, `ft_tolower`

* **String Manipulation:** `ft_strlen`, `ft_strlcpy`, `ft_strlcat`, `ft_strchr`, `ft_strrchr`, `ft_strncmp`, `ft_strnstr`, `ft_strdup`

* **Memory Manipulation:** `ft_memset`, `ft_bzero`, `ft_memcpy`, `ft_memmove`, `ft_memchr`, `ft_memcmp`, `ft_calloc`

* **Conversion:** `ft_atoi`

### Part 2: Additional Functions

Useful utility functions not found in the standard `libc`, aimed at simplifying common parsing and string operations:

* `ft_substr`: Extracts a substring from a given string.

* `ft_strjoin`: Concatenates two strings into a newly allocated string.

* `ft_strtrim`: Trims specified characters from the beginning and end of a string.

* `ft_split`: Splits a string into an array of strings using a specified delimiter character.

* `ft_itoa`: Converts an integer value into a string.

* `ft_strmapi`: Applies a function to each character of a string, creating a new string from the results.

* `ft_striteri`: Applies a function to each character of a string in place.

* **File Descriptor Outputs:** `ft_putchar_fd`, `ft_putstr_fd`, `ft_putendl_fd`, `ft_putnbr_fd`

### Part 3: Linked List Functions

Functions designed to manipulate a basic singly linked list structure defined as `t_list`:

* `ft_lstnew`: Allocates and creates a new node.

* `ft_lstadd_front`: Adds a node to the beginning of the list.

* `ft_lstsize`: Counts the number of nodes in the list.

* `ft_lstlast`: Returns the last node of the list.

* `ft_lstadd_back`: Adds a node to the end of the list.

* `ft_lstdelone`: Deletes and frees a single node's content.

* `ft_lstclear`: Deletes and frees a node and all subsequent nodes.

* `ft_lstiter`: Iterates through the list and applies a function to the content of each node.

* `ft_lstmap`: Applies a function to each node to create a new linked list.

## Instructions

### Compilation

To compile the library, you will need `cc`, `gcc` or `clang` and `make`. Run the following command at the root of the repository:

```bash
make
```

To compile the library including the linked list (bonus) functions, run:

```bash
make bonus
```

Other available rules in the `Makefile`:

* `make clean`: Removes intermediate object files (`.o`).

* `make fclean`: Removes object files and the compiled `libft.a` binary.

* `make re`: Completely recompiles the library.

### Usage

To use this library in your own projects, include the header in your C files:

```c
#include "libft.h"
```

When compiling your project, link the compiled library:

```bash
gcc -Wall -Wextra -Werror your_source_file.c -L. -lft -o your_program
```
